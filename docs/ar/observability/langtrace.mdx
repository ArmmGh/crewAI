# نظرة عامة على Langtrace

Langtrace هو أداة مفتوحة المصدر خارجية تساعدك في إعداد المراقبة والتقييمات لنماذج اللغة الكبيرة (LLMs) وأطر عمل LLM وقواعد بيانات المتجهات.
على الرغم من أنها ليست مبنية مباشرة في LocalAI، يمكن استخدام Langtrace جنباً إلى جنب مع LocalAI للحصول على رؤية عميقة في التكلفة وزمن الاستجابة وأداء وكلاء LocalAI.
يتيح لك هذا التكامل تسجيل المعاملات الفائقة ومراقبة تراجعات الأداء وإنشاء عملية للتحسين المستمر لوكلائك.

## تعليمات الإعداد

<Steps>
   <Step title="التسجيل في Langtrace">
      سجّل بزيارة [https://langtrace.ai/signup](https://langtrace.ai/signup).
   </Step>
   <Step title="إنشاء مشروع">
      عيّن نوع المشروع إلى `LocalAI` وقم بتوليد مفتاح API.
   </Step>
   <Step title="تثبيت Langtrace في مشروع LocalAI">
      استخدم الأمر التالي:

    ```bash
    pip install langtrace-python-sdk
    ```
   </Step>
   <Step title="استيراد Langtrace">
      استورد وهيّئ Langtrace في بداية نصك البرمجي، قبل أي استيرادات LocalAI:

    ```python
    from langtrace_python_sdk import langtrace
    langtrace.init(api_key='<LANGTRACE_API_KEY>')

    # Now import LocalAI modules
    from localagents import Agent, Task, Crew
    ```
   </Step>
</Steps>

### الميزات وتطبيقاتها على LocalAI

1. **تتبع رموز LLM والتكاليف**

   - مراقبة استخدام الرموز والتكاليف المرتبطة لكل تفاعل وكيل LocalAI.

2. **رسم بياني للتتبع لخطوات التنفيذ**

   - تصور تدفق تنفيذ مهام LocalAI، بما في ذلك زمن الاستجابة والسجلات.
   - مفيد لتحديد الاختناقات في سير عمل الوكلاء.

3. **تنظيم مجموعات البيانات مع التعليق اليدوي**

   - إنشاء مجموعات بيانات من مخرجات مهام LocalAI للتدريب أو التقييم المستقبلي.

4. **إدارة إصدارات المطالبات**

   - تتبع الإصدارات المختلفة من المطالبات المستخدمة في وكلاء LocalAI.
   - مفيد لاختبار A/B وتحسين أداء الوكلاء.

5. **ساحة المطالبات مع مقارنات النماذج**

   - اختبار ومقارنة مطالبات ونماذج مختلفة لوكلاء LocalAI قبل النشر.

6. **الاختبارات والتقييمات**

   - إعداد اختبارات آلية لوكلاء ومهام LocalAI.
